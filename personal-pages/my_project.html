Media Intelligent Project (Master Thesis)

The thesis project proposed an algorithm of multivariate LSTM to predict urban passenger flow using the Translink traffic dataset. The algorithm is proved to be valid in three different test cases. The prediction algorithm is deployed on a web-based application using Flask web framework. The application has multi-functional features, which are not limited to traffic flow prediction.

SugarByte (CSRIO)

SugarByte is a series of projects to protect the great barrier reef undertaken by CSRIO. The project utilizes Normalised Difference Vegetation Index (NDVI) data to visualize and forecast sugarcane yield, which is generated by satellite imagery. Based on Google Earth Engine (GEE) platform, our goal is to implement a user-friendly interface to display NDVI data, which aims to help sugarcane farmers manage and monitor crops. In the development team, I am responsible for developing the interface functions requested by our client scholar from CSRIO.
